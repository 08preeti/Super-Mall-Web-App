# 🏬 Super Mall Management System

A web-based mall management system built using HTML, CSS, JavaScript and Firebase.

## 🚀 Features

### 🔐 Authentication
- Role-based login (Admin / User)
- Firebase Authentication
- Access control

### 🏬 Shop Management
- Add / Delete shops
- Assign floor number
- Store location

### 📦 Product Management
- Add products with category and shop
- Dynamic product cards
- Floor-wise, Shop-wise & Category-wise filtering

### 🎁 Offers
- Add discount offers
- Display active offers to users

### 🛒 Cart System
- Add to cart
- Increase / Decrease quantity
- Remove item
- Checkout system
- Revenue tracking

### 🔍 Product Comparison
- Compare 2 products
- Compare category, shop, and price

### 📊 Admin Dashboard
- Total Shops
- Total Products
- Total Offers
- Total Users
- Total Logs
- Total Revenue

---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Firebase Authentication
- Firebase Firestore
